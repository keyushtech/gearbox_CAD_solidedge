# Gearbox_CAD_solidedge
This project, titled "Simple Barbecue Grill," is a collaborative engineering design developed using Solid Edge. It features a comprehensive CAD model of a portable grill assembly, complete with individual part files and structural details. The repository includes the top-level assembly file (`Assembly.asm`) which integrates the main cooking surface (`Grill.par`), the structural foundation (`Base.par`), and various support members like L-brackets and vertical legs (`L-Shape at base.par`, `V.par`, and `LV.par`). Developed using Solid Edge version 226.0, the project utilizes synchronous modeling and features calculated physical properties to assist in structural analysis and fabrication planning.

### File Descriptions

**Assembly.asm**
The primary assembly file that defines the spatial relationships and constraints between all components of the barbecue grill. It serves as the central hub for the design, ensuring all individual parts align correctly for structural integrity and functional use.

**Grill.par**
A detailed CAD model of the main cooking grate, designed with specific spacing for heat distribution and material efficiency. This part represents the functional interface of the grill where the actual cooking takes place.

**Base.par**
The central structural component of the grill body, serving as the housing for the fuel and the support for the upper grill surface. It is designed to withstand high thermal loads while maintaining the overall shape of the unit.

**L-Shape at base.par**
Part file for the structural L-brackets used to reinforce the corners and provide stability to the base assembly. These components are critical for ensuring the grill remains rigid during transport and use.

**V.par & LV.par**
Technical models for the vertical support legs and bracing members, defining the height and load-bearing capacity of the unit. These files detail the geometry required to provide a stable elevated platform for the grill body.

### Technical Details
* **Software:** Solid Edge (Version 226.0)
* **Modeling Approach:** Synchronous Technology
* **Features:** Integrated assembly constraints, physical property data, and fabrication-ready part geometries. This model is suitable for further analysis in finite element software or for direct export to manufacturing workflows.
